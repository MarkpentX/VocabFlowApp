# VocabFlow

**VocabFlow** is a website for learning foreign vocabulary.

It helps users build their own vocabulary, organize words by topics, and practice them using interactive quizzes. The goal of the project is to make vocabulary learning simple, structured, and effective.

## Features

**Save Words**
Add new words and their translations to build your personal vocabulary.

**Organize with Tags**
Group words by topics such as food, travel, business, and more.

**Practice and Learn**
Test yourself with interactive quizzes to reinforce your memory and improve retention.

## Project Status

The project is still under development.

Most of the core functionality is already implemented. The practice system is currently being improved.

## Future Plans

In the future, VocabFlow will include AI-powered features, such as:

* a neural network for practicing English
* automatic generation of word associations to help memorize vocabulary
* smarter learning assistance
